# AI-Powered Fitness & Calorie Tracking System

A fitness tracking application that helps users to monitor daily calorie intake,calorie burned, manage workouts, set fitness goals, and analyze food images using Gemini AI for nutritional insights and personalized health recommendations.

## Features
- User Login & Registration
- Food & Calorie Tracking
- Workout Tracking
- Fitness Goal Management
- AI-Powered Food Image Analysis
- Personalized Health Tips
- Dashboard Overview
- User Profile Management

## Tech Stack
- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript
- Gemini AI
- Maven

## Project Flow

User
↓
Frontend (HTML, CSS, JavaScript)
↓
Spring Boot REST API
↓
Gemini AI / MySQL Database
↓
Dashboard & Analytics

## Key Highlights
- Built a full-stack fitness tracking application using Java, Spring Boot, and MySQL.
- Integrated Gemini AI to analyze food images and estimate nutritional information.
- Developed REST APIs for managing food logs, workouts, fitness goals, and user profiles.
- Generated personalized health tips based on calorie intake, workout activity, and  fitness goals.
- Designed an interactive dashboard to monitor fitness activities and daily calorie consumption.
